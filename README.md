# CryptoPlayer - 加密视频播放器

一个"壳加密视频播放器"，将版权提示视频与加密的正片视频合并成一个文件，只有使用特定播放器才能播放完整内容。<br>
本项目源于一次好奇，想制作一个加密播放器，于是和ChatGPT讨论构思、由deepseek创建程序功能设计及初始版本、由chatglm进行调试和后续优化<br>

## 快速开始

```bash
# 1. 安装依赖
pip install pycryptodome

# 2. 运行交互式工具（推荐）
python interactive_tool.py

# 3. 按照菜单提示操作即可
```

**Windows用户**：无需安装FFmpeg！项目包含内置的FFmpeg组件，开箱即用。<br>
**Linux/macOS用户**：需要安装FFmpeg并添加到PATH环境变量。

## 核心功能

- ✅ **通用播放器兼容**：任何播放器都可以播放提示段（版权提示）
- ✅ **加密保护**：正片内容使用AES加密，只有本程序可以解密播放
- ✅ **自动屏幕适配**：高分辨率视频自动缩放以适应屏幕
- ✅ **交互式工具**：友好的命令行交互界面
- ✅ **通用文件加密**：支持加密任意文件类型（zip、txt、图片等）
- ✅ **内置FFmpeg**：Windows用户开箱即用，无需环境配置

## 常见问题

**Q: 通用播放器能播放加密文件吗？**
A: 可以播放文件的提示段（前10秒左右的版权提示），但无法播放加密的正片内容。

**Q: 如何自定义提示视频？**
A: 运行 `python interactive_tool.py`，在加密时选择提示视频文件即可。

**Q: 视频分辨率太高超出屏幕？**
A: 已修复。播放器会自动检测屏幕尺寸，如果视频超出屏幕会自动缩放并保持宽高比。

**Q: 如何加密zip、pdf等非视频文件？**
A: 运行 `python interactive_tool.py`，选择"其他文件（用于隐写术）"类别的文件。
